# Sales-Tech-data-cleaning-and-preparation
This is the dataset of a hypothetical sales company that deals with gadgets
The dataset contains the following columns;
OrderID, customerID, salesRepID, SalesRepName, Date, Time, ProductName, ProductCategory, Quantity, unitPrice, SalesRegion, SalesRep emailaddress, feedback, etc
Tools/methods used in cleaning the data include: TRIM, CLEAN, PROPER, VLOOKUP, FILTER
VLOOKUP was used to calculate the tax rate and the discount rate
Calculated gross revenue, total cost, tax amount, discount amount, net sales, profit/loss using Excel functions.
